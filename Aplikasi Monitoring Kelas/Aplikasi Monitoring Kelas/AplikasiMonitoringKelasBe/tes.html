üîê Security Architecture & Best Practices

üåê Edge Security

  • AWS WAF: Web Application Firewall untuk filtering malicious traffic, SQL injection, XSS protection
  • AWS Shield Standard: DDoS protection otomatis gratis
  • CloudFront: HTTPS enforcement, custom SSL certificates via ACM
  • Route 53: DNSSEC untuk DNS security, health checks

üè¢ Network Security

  • VPC Isolation: Private network dengan subnet segregation (public, web, app, database)
  • Security Groups: Stateful firewall di level instance, whitelist port & IP
  • Network ACLs: Stateless firewall di level subnet untuk additional layer
  • NAT Gateway: Outbound internet access untuk private subnets tanpa expose instance
  • VPC Flow Logs: Network traffic monitoring untuk audit & troubleshooting

üîí Data Encryption

  • In Transit: TLS/SSL via ACM certificates untuk semua communication
  • At Rest: EBS volumes encrypted dengan AWS KMS, RDS encryption enabled
  • S3 Encryption: Server-Side Encryption (SSE-S3 atau SSE-KMS)
  • ElastiCache: Encryption in-transit dan at-rest dengan Redis AUTH

üë§ Identity & Access Management

  • IAM Roles: EC2 instance roles dengan least privilege principle
  • IAM Policies: Fine-grained permissions untuk setiap resource
  • MFA: Multi-Factor Authentication untuk root dan admin accounts
  • CloudTrail: API call logging untuk audit compliance
  • AWS Config: Resource configuration tracking & compliance monitoring

üóÑÔ∏è Database Security

  • Private Subnets: RDS hanya accessible dari application tier
  • Encryption: RDS encryption at rest dengan KMS keys
  • Automated Backups: Daily automated snapshots dengan encryption
  • Read Replicas: Di separate AZ untuk high availability
  • Database Activity Streams: Real-time monitoring untuk compliance

üìä Monitoring & Logging

  • CloudWatch: Metrics, logs, dan alarms untuk proactive monitoring
  • CloudWatch Logs: Centralized logging dari semua services
  • AWS X-Ray: Distributed tracing untuk performance debugging
  • GuardDuty: Threat detection service untuk malicious activity
  • Security Hub: Centralized security findings dari multiple services
üîÑ Request Flow & Data Journey
1
User Request
Pengguna mengirimkan permintaan melalui browser atau aplikasi, yang memulai alur dengan resolusi DNS. Fitur ini mencakup akses cepat ke konten statis atau dinamis tanpa penjelasan spesifik dari gambar, tetapi mendukung integrasi dengan layanan seperti CloudFront untuk distribusi global.
‚Üí
2
Route 53
DNS Resolution
Route 53 menyelesaikan nama domain ke IP address, mengarahkan lalu lintas ke layanan yang tepat. Fitur ini mendukung routing lalu lintas berbasis latency, geo-location, dan health checks untuk ketersediaan tinggi, meskipun tidak secara langsung disebutkan di gambar.
‚Üí
3
WAF
Security Check
AWS WAF memeriksa dan memfilter lalu lintas masuk untuk mencegah serangan seperti SQL injection atau XSS. Fitur ini memberikan perlindungan di tingkat edge, bekerja sama dengan CloudFront untuk keamanan awal sebelum mencapai backend.
‚Üí
4
CloudFront
CDN Cache
Sesuai gambar, Amazon CloudFront (CDN) menggunakan edge caching untuk mendistribusikan konten PVA agar lebih cepat diakses dari lokasi terdekat user. Ini mengurangi latency, meningkatkan performance, dan membuat loading time lebih cepat dibandingkan dari data center utama. CDN managed dasar dengan 3 opsi, memungkinkan manajemen mandiri di area Indonesia untuk efisiensi biaya dan kecepatan.
‚Üí
5
ALB
Load Balance
Application Load Balancer mendistribusikan lalu lintas ke instance EC2 yang sehat. Fitur ini mendukung path-based routing, HTTPS termination, dan integrasi dengan Auto Scaling untuk menangani lonjakan lalu lintas, melengkapi serverless options seperti Lambda dari gambar.
‚Üí
6
EC2 Web
Processing
EC2 instances di web tier memproses permintaan front-end. Untuk alternatif serverless sesuai gambar, AWS Lambda (Serverless) menyediakan auto-scaling serverless backend yang autentikasi, grading, dan quick deploy on-demand. Ini hemat biaya dengan pay-per-execution, mendukung hingga 95% penghematan biaya dibandingkan server tradisional.
‚Üí
7
EC2 App
Business Logic
EC2 instances di app tier menangani logika bisnis. Integrasi dengan Amazon Bedrock (AI) dari gambar memungkinkan foundation model Claude 2 Haiku, memberi kemampuan AI generatif seperti text generation, summarization, dan Q&A dengan biaya rendah sekitar $0.00025 per 1K token input/output.
‚Üí
8
Cache/Database
Data Retrieval
Data diambil dari cache atau database. Sesuai gambar, Amazon DynamoDB sebagai NoSQL database dengan single-digit millisecond latency, on-demand pricing hanya charge per request, auto scaling, built-in security, backup, caching untuk handle millions of requests. ElastiCache (Redis) untuk in-memory caching layer dengan response <1ms, memungkinkan caching data populer untuk mengurangi beban database. Integrasi S3 + CloudFront untuk storage statis compressed unik, biaya $0.023/GB per bulan untuk jutaan file.
